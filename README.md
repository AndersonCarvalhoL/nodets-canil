># nodets-canil
![Preview](/.github/Preview.png)

Projeto desenvolvido em Node + Typescript

>### Tecnologias

- HTML
- CSS
- Typescript
- Nodejs
- Express
- Mustache
- Git e GitHub


>### Pré-requisitos globais:
`npm i -g nodemon typescript ts-node`

>### Instalação
`npm install`

>### Para rodar o projeto
`npm run start-dev`
